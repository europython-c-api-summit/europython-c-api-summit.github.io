The Python C API has many different stakeholders, including but not limited to CPython core developers, maintainers of Python extensions, code generators (Cython, pybindy11, etc) and alternative APIs (HPy).

The goal of this summit is to make sure that people from all these different projects can meet and discuss the state of the C API, existing challenges and ongoing work.

The Python C API summit is intended to be a one day event planned to take place as part of the EuroPython 2024 conference in Prague between July 8 to July 14 2024. You will need to have a valid EuroPython in-person ticket (Conference, Tutorial or Combined) to participate.

The number of participants may be limited. Please register using the following form. If you are intereted in giving a presentation, please include a short abstract in the registration form.
